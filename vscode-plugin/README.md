# vscode-plugin

`vscode-plugin` 是 `tdodoapp` 的 VS Code 插件项目：它通过 `tdodoapp` 的公开包入口启动开发服务。

## 当前版本

- `0.2.0`
- 插件中心与活动栏统一使用紫色 `<>` 图标，避免与 VS Code 内置图标混淆。
- 打开抽屉时插件启动内置静态服务；隐藏、关闭抽屉或退出 VS Code 时停止服务。启动成功后，抽屉标题显示实际监听端口，描述显示完整地址。

## 更新与任务台账

此文件记录插件的公开版本说明。项目任务的编号、状态和验证结论只维护在项目根目录 `README.md` 的 `任务台账` 中；更新插件后，以该台账的对应任务为准，不在此处复制第二份台账。
